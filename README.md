# 🌿 Heritage Estate — The Art of Single Origin Tea

![Heritage Estate Hero Banner](public/heritage-hero.png)

> **"A symphony of tradition and technology, bringing the finest single-origin tea directly from the estates of Sivasagar to your digital doorstep."**

Welcome to **Heritage Estate**, a premium digital experience crafted for tea connoisseurs. This project is not just a catalogue; it's an immersive journey powered by cutting-edge web technologies, designed to celebrate the rich heritage of tea cultivation in North East Assam.

---

## ✨ Key Features

- **🌐 Immersive 3D Interactions**: Integrated [Spline](https://spline.design/) models allow users to interact with physical representations of our heritage.
- **🎬 Fluid Animations**: Powered by [GSAP (GreenSock)](https://gsap.com/), every transition, scroll, and hover is meticulously choreographed for a premium feel.
- **🍵 Dynamic Catalogue**: A full-featured viewing experience for our seasonal flushes, from bold Second Flush Assams to artisan Bamboo-aged rarities.
- **📱 Ultra-Modern Design**: Built using the latest [Tailwind CSS v4](https://tailwindcss.com/), ensuring a lightning-fast, responsive, and visually stunning interface across all devices.
- **⚡ Reactive Performance**: Leverages [React 19](https://react.dev/) and [Vite](https://vitejs.dev/) for near-instant load times and seamless state management.

## 📁 Engineering Blueprint

```bash
Heritage-Tea/
├── public/                 # Static assets (3D models, high-res images)
├── src/
│   ├── components/         # Reusable UI building blocks
│   │   ├── 3d/             # Spline scene integrations
│   │   ├── layout/         # Header, Footer, and Page wrappers
│   │   ├── sections/       # Modular page sections (Hero, Featured, etc.)
│   │   └── ui/             # Atomic design elements (Buttons, Inputs)
│   ├── data/               # Project configuration and tea catalogue (Single Source of Truth)
│   ├── hooks/              # Custom React hooks for animations and state
│   ├── Pages/              # Main route entries (Home, Collection, About, Contact)
│   ├── App.jsx             # Root component and Routing logic
│   ├── index.css           # Global styles and Tailwind 4 layers
│   └── main.jsx            # Application entry point
├── vite.config.js          # Vite optimization & plugin config
└── package.json            # Dependency manifest
```

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/rumman2004/Heritage-Tea.git
cd Heritage-Tea
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Launch Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
